# MedSkills - Фронтенд для приложения спортивного врачевания

## 📋 О проекте

**MedSkills** — это веб-приложение для спортивных врачей, разработанное в рамках производственной практики. Приложение предоставляет платформу для тестирования знаний в области спортивной медицины.

### 🎯 Задание на практику

> Создание фронтенда для приложения спортивного врачевания с использованием Vue.js (Nuxt)

## ✨ Функциональность

### 🔐 Профиль пользователя

- Просмотр и редактирование личных данных (ФИО, email, телефон, специализация)
- Модальное окно для редактирования профиля

### 📝 Тестирование

- **4 категории тестов:**
  - 🦴 Травматология спортивная
  - ❤️ Спортивная кардиология
  - 🏋️ Реабилитация после травм
  - 🥗 Питание спортсменов
- **12 тестов** (по 3 в каждой категории)
- Сохранение прогресса тестирования
- Возможность продолжить незавершённый тест
- Отображение результатов после завершения

### 📊 Статистика

- Общая статистика по всем тестам
- Статистика по категориям
- Отображение последних результатов
- Процент выполнения и средний балл

### 📚 База знаний

- Статьи по спортивной медицине
- Фильтрация по категориям
- Поиск по статьям
- Популярные статьи
- Модальное окно с полным содержанием статьи

### ⚙️ Настройки

- Включение/отключение уведомлений
- Выбор языка интерфейса (8 языков)
- Переключение тёмной/светлой темы
- Сброс всех локальных данных

### 🌍 Локализация

Поддерживается **8 языков**:

### 🔔 Уведомления

- Всплывающие уведомления о действиях
- История уведомлений в шапке
- Счётчик непрочитанных уведомлений

### 🎨 Темы оформления

- Светлая тема

### 🆘 Помощь

## 🚀 Запуск проекта

### Предварительные требования

- **Node.js** версии 20.19.0 или выше (рекомендуется 22.12.0+)
- **npm** или **yarn**

### Установка и запуск

# 1. Клонируйте репозиторий

git clone https://github.com/TWOKIIII/medskills-frontend.git

# 2. Перейдите в папку проекта

cd medskills-frontend

# 3. Установите зависимости

npm install

# 4. Запустите проект в режиме разработки

npm run dev

# 5. Откройте браузер и перейдите по адресу

# http://localhost:3000

### Доступные скрипты

| Команда            | Описание                        |
| ------------------ | ------------------------------- |
| `npm run dev`      | Запуск в режиме разработки      |
| `npm run build`    | Сборка проекта для продакшена   |
| `npm run generate` | Генерация статического сайта    |
| `npm run preview`  | Предпросмотр собранного проекта |

### Очистка кэша (если возникают проблемы)

# Удаление папок кэша

rm -rf .nuxt .output node_modules/.cache

# Для Windows (PowerShell)

Remove-Item -Recurse -Force .nuxt, .output -ErrorAction SilentlyContinue

# Перезапуск

npm run dev
